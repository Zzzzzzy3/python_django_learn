

### **食堂后台管理系统流程图**

#### **1. 用户登录流程**



```plaintext
用户登录
  │
  ├─ 输入用户名和密码
  │
  └─ 验证用户身份
       │
       ├─ 验证成功 → 进入主界面
       │
       └─ 验证失败 → 提示错误信息，返回登录界面
```

#### **2. 主界面功能选择**



```plaintext
主界面
  │
  ├─ 账户管理
  │
  ├─ 交易记录管理
  │
  ├─ 菜品管理
  │
  ├─ 库存管理
  │
  ├─ 供应商管理
  │
  └─ 采购记录管理
```
